## React wrapper & deploy

Теперь проект снабжён простой оболочкой на React (Vite), которая показывает оригинальные HTML-страницы внутри `iframe` и делает проект готовым для деплоя как React-приложение.

Запуск локально:

```bash
npm install
npm run dev
```

Сборка и деплой (`gh-pages`):

```bash
npm run deploy
```

Если нужен полный перевод всех страниц в JSX (React-компоненты), могу выполнить этот шаг дополнительно.
